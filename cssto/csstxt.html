Color

You can set the color of text with the following: color: value; Possible values are

  • color name – example:(red, black…)
  • hexadecimal number – example:(#ff0000, #000000)
  • RGB color code – example:(rgb(255, 0, 0), rgb(0, 0, 0))

    • Letter Spacing

    You can adjust the space between letters in the following manner. Setting the value to 0, prevents the text from justifying. You can use negative values. letter-spacing: value; Possible values are

  • normal
  • length

    • Text Align

    text-align: value; Possible values are

  • left
  • right
  • center
  • justify
