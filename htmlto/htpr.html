HTML | Paragraphs

The <p> tag is HTML defines a paragraph.
These have both opening and closing tag.

<p> This is a paragraph. </p>

Example

<html>
<head>
<!--Information about the page-->
     <title> Basic Coding 101 </title>
</head>
<body>
<!--Contents of the webpage-->
</body>
</html>

Click on the button to copy the text from the text field.

Break Line

These is a way to let the HTML know where does the browser need to change the lines by the use of <br> tag.
These tags do not have any closing tag. So just a single opening tag will change the line

<br>

<!DOCTYPE html>
<html>
<head>
<!--Information about the page-->
     <title> Let's Code </title>
</head>
<body>
<!--Contents of the webpage-->
     <p> This is a paragraph 1. </p> <br>
     <p> This is a paragraph 2. </p> <br>
</body>
</html>

<pre> Element

The HTML <pre> element defines preformatted text.
The text inside a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.

<pre>
Yeah, I'm gonna take my horse to the old town road

I'm gonna ride 'til I can't no more

I'm gonna take my horse to the old town road

I'm gonna ride 'til I can't no more
</pre>
