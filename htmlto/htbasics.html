HTML | Basics



Don't worry if these examples use tags you have not learned.
You will learn about them in the next section.

In this section, we will go through all the basic stuff of HTML coding. There are various tags that we must consider and include while starting to code in HTML. These tags helps in rganization and basic formatting of elements in our script or web pages. These step by step procedures will guide you through the process of writing HTML.

Basic HTML Document

Every HTML document begins with a HTML document tag. Although this is not mandatory but it is a good convention to start the document with this below mentioned tag

<!DOCTYPE html>

Every HTML code must be enclosed between basic HTML tags.
It begins with <html> and ends with </html>

<html>....</html>

The head tag comes next which contains all the header information of the web page or document like the title of the page and other miscellaneous information.
These informations are enclosed within head tag which opens with
<head> and ends with </head>
The contents will of this tag will be explained in the later sections of course.

<head>....</head>

We can mention the title of a web page using the <title> tag.
This is a header information and hence mentioned within the header tags. The tag begins with <title> and ends with </title>

<title>Let's Code</title>

This is the most important of all the tags we have learned so far. The body tag contains the actual body of the page which will be visible to all the users.
This opens with <body> and ends with </body>
Every content enclosed within this tag will be shown on the web page be it writings or images or audios or videos or even links. We will see later in the section how using various tags we may insert mentioned contents into our web pages.
